# Category 09 — Radio, Comms & Sensors

Enclosures, mounts, and accessories for Meshtastic, amateur radio,
weather stations, and remote sensors.

## Notes for Meshtastic / LoRa Nodes
- Use PETG minimum for any outdoor Heltec V3 enclosure
- UV-stable black PETG runs ~10°C warmer in direct sun — consider natural/white
- Seal cable entry points with silicone; the clip itself is not IP-rated
- Models suitable for Alberta outdoor node deployment (rated to −40°C in PETG)

## Included STL Files
- `sma_cap_organizer_tray.stl` — 3-compartment SMA dust-cap tray (bench organizer)
- `sma_pigtail_strain_relief.stl` — Strain-relief bracket for SMA pig-tails
