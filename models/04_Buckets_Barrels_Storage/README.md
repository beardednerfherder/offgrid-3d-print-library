# Category 04 — Buckets, Barrels & Storage

Accessories for 5-gallon buckets, IBC totes, Gamma Seal lids, and general
homestead storage.

## Material Recommendation
PETG for most applications. ASA for outdoor barrel fittings exposed to direct sun.

## Included STL Files
- `bucket_rim_label_clip.stl` — J-hook label holder for standard bucket rim
- `screw_organizer_tray_80x50.stl` — Open-top parts organizer tray

## Community Models
See `manifests/models.csv` — filter `category_num = 04`.
Includes: Gamma Seal handles, spigot drill guides, IBC tote strainers.
